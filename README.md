# QR Code Generator (Python)

A simple Python project that generates QR codes from text or URLs.  
This project uses the `qrcode` library to create QR code images.

## Features
- Generate QR codes from text, numbers, or URLs
- Save QR code as PNG file
- Advanced version with custom colors and size
